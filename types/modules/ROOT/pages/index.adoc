Общий балл за курс выставляется по следующей формуле:

* Теоретические домашние задания — 30%
* Проектные задания — 50%
* Устный экзамен — 20%

Пороги для оценки по пятибальной шкале:

* ≥ 90% — отлично
* ≥ 75% — хорошо
* ≥ 60% — удовлетворительно
* < 60% — неудовлетворительно
